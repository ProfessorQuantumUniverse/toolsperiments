# toolsperiments 🧪

Eine gesammelte Spielwiese kleiner Web-Tools, Demos und Experimente — Dinge, die kein eigenes Repo brauchen. Jedes Unterverzeichnis ist ein eigenständiges Mini-Projekt, das über die zentrale **[Landingpage](https://professorquantumuniverse.github.io/toolsperiments/)** erreichbar ist.

## Enthaltene Experimente

| Experiment | Ordner | Worum geht's |
|---|---|---|
| 🌌 Interstellar | [`interstellar/`](./interstellar/) | Immersive 3D-Reise mit Gargantua- & Wurmloch-Shadern |
| 🌞 Aurora Voyager | [`aurora-cycles/`](./aurora-cycles/) | Sonnenzyklus & Live-Tracking von Polarlichtern |
| 🌍 Sci-Fi Global Monitor | [`cool-globe/`](./cool-globe/) | 3D-Globus-Experiment mit globe.gl |
| π π Day | [`pi-day/`](./pi-day/) | Countdown, 1000 Stellen von π & Pi-Spielereien |
| 🧠 Weisheit Einsteins | [`einstein-wisdom/`](./einstein-wisdom/) | Zitatesammlung von Albert Einstein |
| 🎄 Aggressive Christmas | [`aggressive-christmas/`](./aggressive-christmas/) | Aggressiver Weihnachts-Countdown |
| 🍌 Minion World | [`minion-world/`](./minion-world/) | Das verrückteste Minion-Fandom |
| 💻 Hacking Simulator | [`hacking-simulator/`](./hacking-simulator/) | Spaßiger „Ultra-Realism"-Hacking-Simulator |
| 🤖 AI-Abo-Vergleich | [`AiComparison/`](./AiComparison/) | Entscheidungshilfe für KI-Abos |

## Aufbau

```
toolsperiments/
├── index.html        ← Landingpage (verlinkt alle Experimente)
├── <experiment>/     ← je ein Unterordner pro Mini-Projekt
└── LICENSE
```

Neues Experiment hinzufügen: Ordner anlegen, `index.html` hineinlegen, eine Karte in die Wurzel-`index.html` ergänzen — fertig.
